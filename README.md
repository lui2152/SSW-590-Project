# SSW-590-Project: FlashClash
FlashClash is a web-based platform that acts as a bridge between passive and active learning  

Two Modes: Study Mode and Clash Mode
1. Study Mode: Study all flashcards until completion 
2. Clash Mode: Fight against a bot on a timer with a score system on a randomized set of 10 flashcards, be the first to answer or you'll lose the point !!

App Hosted On: http://3.19.75.155:3000/

# Prerequisites for Local Host Launch
1. Install Docker Desktop 
2. Install Git 
3. Clone the Repository (git clone https://github.com/lui2152/SSW-590-Project.git)

# Local Host Launch 
1. Open Docker Desktop
2. Open terminal and type: docker compose build
3. After building, type: docker-compose up -d
4. Open http://localhost:3000/ to access application
5. NOTE: To access dashboards (Prometheus and Grafana), go to http://localhost:4000/
