# SSW-590-Project: FlashClash
FlashClash is a web-based platform that acts as a bridge between passive and active learning  

Two Modes: Study Mode and Clash Mode
1. Study Mode: Study all flashcards until completion 
2. Clash Mode: Randomized set of 10 flashcards are tested with a timer and a score system 

Group Members: Ivana Lu, Keona Hicks, Jessica Militello, Hemang Shankar, Sabrina Trestin  

Instructions for launch:
Prerequisites: Docker Desktop 
1. Open terminal and build using docker-compose build
2. After building, type docker-compose up -d to run 
3. Open http://localhost:3000/
